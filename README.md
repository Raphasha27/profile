## 💼 Featured: SmartBank Enterprise Platform

[![SmartBank](https://img.shields.io/badge/SmartBank-Enterprise-blueviolet?style=for-the-badge)](https://github.com/Raphasha27/smartbank-enterprise-platform)
[![Deploy](https://img.shields.io/badge/Deploy_to-Render-46E3B7?style=for-the-badge&logo=render&logoColor=white)](https://render.com/deploy?repo=https://github.com/Raphasha27/smartbank-enterprise-platform)

Distributed banking backend — 8 microservices (Auth, Account, Transaction, Loan, Audit, Notification, Ledger, Gateway) with Kafka event bus, saga pattern, optimistic locking, and OpenTelemetry tracing.

```
Client → API Gateway → Auth → Transaction → Account → Kafka → Fraud + Audit + Ledger
```

**Architecture:** Microservices · Event-Driven · API Gateway · CQRS-adjacent  
**Data:** PostgreSQL per service · ACID · Optimistic Locking · Idempotency  
**Infra:** Docker · Docker Compose · Render Blueprint · GitHub Actions CI
